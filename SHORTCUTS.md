# OV 단축키
{: .no_toc }

앱 안에서 `⌘/`(Windows/Linux: `Ctrl+/`)로 같은 시트를 띄울 수 있습니다.

  <summary>카테고리</summary>

* TOC
{:toc}
</details>


## 내비게이션

| 단축키 | 동작 |
|---|---|
| `⌘K` | 명령 팔레트 |
| `⌘P` | 파일 빠른 점프 |
| `⌘B` | 사이드바 열기/닫기 |
| `⌘D` | 오늘의 노트 |
| `⌘,` | 설정 열기 |

## 뷰 전환

| 단축키 | 동작 |
|---|---|
| `⌘1` | 에디터 모드 |
| `⌘2` | 데이터베이스 모드 |
| `⌘⇧G` | 그래프 오버레이 |
| `⌘⇧L` | 외부 링크 오버레이 |
| `⌘I` | 정보 패널 (속성 / 백링크) |

## 에디팅

| 단축키 | 동작 |
|---|---|
| `/` | 슬래시 메뉴 (블록 삽입) |
| `[[` | 위키링크 자동완성 |
| `⌘B` | 굵게 |
| `⌘I` | 기울임 |
| `⌘S` | 현재 노트 저장 |
| `Tab` (목록) | 들여쓰기 |
| `⇧Tab` (목록) | 내어쓰기 |

## AI Copilot

| 단축키 | 동작 |
|---|---|
| `⌘J` | Copilot 열기/닫기 |
| `⌘↵` | 메시지 전송 |
| `⇧↵` | 메시지 내 줄바꿈 |

## 도움말

| 단축키 | 동작 |
|---|---|
| `⌘/` | 단축키 시트 |
| `?` (우측 레일) | 도움말 메뉴 |

## 캘린더 / 데이터베이스

| 단축키 | 동작 |
|---|---|
| 캘린더 그리드 클릭 | 그 날 선택 |
| Day view에서 `←` / `→` | 전날/다음날 |
| Kanban 카드 드래그 | 다른 열로 속성 변경 |
| Table 헤더 클릭 | 정렬 |

## 일반

| 단축키 | 동작 |
|---|---|
| `Esc` (모달 안에서) | 닫기 |
| `Tab` / `⇧Tab` (모달 안에서) | 포커스 순환 |
| `⌘Q` | 종료 (저장 안 한 변경 있으면 확인 다이얼로그) |
| `⌘R` | 페이지 리로드 (저장 안 한 변경 있으면 확인) |

> Windows/Linux에서는 `⌘`를 `Ctrl`로 읽으세요. 일부 키(`⌘⇧G`, `⌘⇧L`)는 OS가 기본 단축키로 예약해서 일부러 `Shift`를 추가한 chord입니다.
